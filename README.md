# 🔐 Password Strength Checker

A simple Python-based tool to **evaluate password strength** and give
friendly suggestions on how to make passwords harder to crack.

> ⚠️ This is for learning and demo purposes only.  
> Do **not** paste real passwords here.

---

## 🚀 Features

- Scores password strength on multiple factors
- Checks:
  - Length
  - Uppercase / lowercase usage
  - Digits
  - Special characters
  - Common weak patterns
- Gives human-readable suggestions to improve weak passwords
- Simple CLI interface (works in any terminal)

---

## ▶️ How to Run

```bash
python password_strength_checker.py
